Project: Software Piracy Control by Using Secured Key Generation

Software piracy control using secured key generation is automation software for generating product key/serial key based on encryption. A serial key is generated based on the processor id and hard disk id, and therefore is completely machine dependent. When software is tried to be cracked illegally, it tries to authenticate the provided IDs. Since the processor id and hard disk id are unique, it can’t be made to work in any machine other than the registered machine. This is so that the software cannot be installed on other computers after it has been installed on one.
